# neovim config

My configuration files for neovim

## Installation
`mkdir ~/.config/nvim`  
`cd ~/.config/nvim`  
`git clone https://github.com/jules-franko/neovim.git . && rm -rf ~/.config/nvim/.git && nvim`  

## Removal
`rm -rf ~/.config/nvim && rm -rf ~/.local/share/nvim && rm -rf ~/.local/state/nvim`
